# Assignment 04 - Used Sales Marketplace

## Aim

Develop a marketplace application for listing, viewing, updating, and deleting used products.

## Technologies Used

Node.js, Express, EJS, MongoDB, Mongoose, Multer, Method Override, HTML, and CSS.

## Run Locally

1. Open `source-code` in a terminal.
2. Run `npm install`.
3. Configure a MongoDB connection if required by the application.
4. Run `node app.js`.

## Output Proof

Add screenshots of the marketplace screens to `output-screenshots` before final submission.
