# Assignment 03 - RESTful CRUD Application

## Aim

Implement a RESTful CRUD application with Express and EJS templates.

## Technologies Used

Node.js, Express, EJS, UUID, Method Override, HTML, and CSS.

## Run Locally

1. Open `source-code` in a terminal.
2. Run `npm install`.
3. Run `node index.js`.

## Output Proof

Add screenshots of the application routes and CRUD operations to `output-screenshots` before final submission.
