# Assignment 01 - Blog Application

## Aim

Build a blog application that supports creating, viewing, editing, and deleting posts.

## Technologies Used

Node.js, Express, EJS, MongoDB, Mongoose, and CSS.

## Run Locally

1. Open `source-code` in a terminal.
2. Run `npm install`.
3. Configure a MongoDB connection if required by the application.
4. Run `node app.js`.

## Output Proof

The blog post creation page is captured in `output-screenshots/blog-create-post.png`.
